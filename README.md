# Dotfiles

Nothing special, just my dotfiles.

## Installation

These are managed by [Dotter](https://github.com/SuperCuber/dotter), a simple dotfile manager.

1. Install `dotter`
2. Clone this repo
3. Run `dotter deploy` inside the repo
4. Profit!?
